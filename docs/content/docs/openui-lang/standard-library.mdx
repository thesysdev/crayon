# Using the Standard Library

OpenUI comes with a **"Batteries Included"** Standard Library containing 40+ pre-built components (Charts, Forms, Layouts). This is the fastest way to get started.

### Installation

The standard library is part of the `react-ui` package, but the logic lives in `react-headless`.

```bash
npm install @openuidev/react-ui @openuidev/react-headless
```

### Usage

You don't need to define any schemas. Just import `standardLibrary` and pass it to the Renderer.

```tsx
import { Renderer } from '@openuidev/react-headless';
import { standardLibrary } from '@openuidev/react-ui';

// In your chat component
<Renderer 
  library={standardLibrary} 
  text={streamedText} 
/>
```

### Generating the Prompt

To make the LLM "aware" of these components, you need to inject the system prompt. The standard library comes with a pre-generated prompt and examples.

```typescript
// app/api/chat/route.ts
import { standardLibrary } from '@openuidev/react-ui';

export async function POST(req: Request) {
  // 1. Get the system prompt
  const systemPrompt = standardLibrary.prompt({
    includeExamples: true // Adds "Show me a chart" -> "BarChart(...)" examples
  });

  // 2. Send to LLM
  const response = await openai.chat.completions.create({
    model: "gpt-4o",
    messages: [
      { role: "system", content: systemPrompt },
      ...req.body.messages
    ],
    stream: true
  });
  
  // ... handle stream
}
```

### Extending the Library

Ideally, you want to mix the Standard Library with your own domain-specific components (e.g., a `ProductCard` or `FlightTicket`).

You can merge libraries easily:

```typescript
import { createLibrary } from '@openuidev/react-headless';
import { standardLibrary } from '@openuidev/react-ui';

const myLibrary = createLibrary({
  name: "Hybrid Library",
  // Merge standard components with your own
  components: {
    ...standardLibrary.components,
    // Your custom components go here
  }
});
```
