## Why a new language?

Most AI tools force you to choose between **Text** (easy but unstructured) and **JSON** (structured but brittle/slow). OpenUI Lang is built specifically for the way LLMs generate tokens.

<div className="grid md:grid-cols-3 gap-6 my-8">

      ⚡️ Streaming First
    </div>
    <p className="text-sm text-slate-600 dark:text-slate-400">
      Line-oriented syntax means the UI renders line-by-line. No waiting for valid JSON closing braces.
    </p>

      📉 Token Efficient
    </div>
    <p className="text-sm text-slate-600 dark:text-slate-400">
      Uses ~50% fewer tokens than JSON-based streaming formats, significantly reducing inference cost and latency.
    </p>
  </div>

  <div className="p-6 border border-slate-200 dark:border-slate-800 rounded-xl bg-slate-50 dark:bg-slate-900/40">

    </div>
    <p className="text-sm text-slate-600 dark:text-slate-400">

    </p>

## How it works

OpenUI Lang acts as the bridge between your **Zod Definitions** and your **React Components**.

<div className="grid md:grid-cols-2 gap-8 my-12">
  
  {/* Step 1 */}
  <div className="space-y-3">
    <div className="flex items-center gap-3">
      <div className="flex items-center justify-center w-8 h-8 rounded-full bg-blue-100 text-blue-600 font-bold text-sm">1</div>
      <h3 className="text-lg font-semibold m-0">You Define (Zod)</h3>
    </div>
    <p className="text-sm text-slate-600 dark:text-slate-400">
      Describe your component props using standard Zod schemas.
    </p>
    <div className="relative">
      ```typescript
      const Chart = z.object({
        title: z.string(),
        data: z.array(z.number())
      });
      ```
    </div>
  </div>

  {/* Step 2 */}
  <div className="space-y-3">
    <div className="flex items-center gap-3">
      <div className="flex items-center justify-center w-8 h-8 rounded-full bg-purple-100 text-purple-600 font-bold text-sm">2</div>
      <h3 className="text-lg font-semibold m-0">We Generate (Prompt)</h3>
    </div>
    <p className="text-sm text-slate-600 dark:text-slate-400">
      OpenUI converts this into a compact system instruction.
    </p>
    <div className="relative">
      ```text
      // System Prompt (Auto-generated)
      Component: Chart(title: string, data: number[])
      ```
    </div>
  </div>

  {/* Step 3 */}
  <div className="space-y-3">
    <div className="flex items-center gap-3">
      <div className="flex items-center justify-center w-8 h-8 rounded-full bg-amber-100 text-amber-600 font-bold text-sm">3</div>

    <p className="text-sm text-slate-600 dark:text-slate-400">
      The model responds in OpenUI Lang, a concise declarative syntax.

      ```text
      root = Root([chart])
      chart = Chart("Q3 Revenue", [10, 20, 30])
      ```
    </div>
  </div>

  <div className="space-y-3">
    <div className="flex items-center gap-3">
