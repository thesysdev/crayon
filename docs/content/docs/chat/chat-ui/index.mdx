OpenUI Chat UI gives you a complete chat experience out of the box, with the flexibility to customize every part of it.

## Chat components

Three ready-to-use components for different use cases:

<Cards>
  <Card title="Copilot" href="/docs/chat/chat-ui/copilot">
    A sidebar chat panel that sits alongside your app.
  </Card>
  <Card title="FullScreen" href="/docs/chat/chat-ui/full-screen">
    A full-page standalone chat experience.
  </Card>
  <Card title="BottomTray" href="/docs/chat/chat-ui/bottom-tray">
    A floating chat widget anchored to the bottom of the page.
  </Card>
</Cards>
