# Chat

Crayon's chat packages give you a complete chat experience out of the box:

- **`useChat`** — a single hook that manages messages, threads, streaming, and LLM communication
- **`CrayonChat`** — a pre-built UI component with multiple layouts, theming, and branding

## How it fits together

```
useChat (state + logic)  →  CrayonChat (UI)
```

`useChat` handles all the state management. `CrayonChat` renders it. They connect via a single `chat` prop.

```tsx
const chat = useChat({ apiUrl: "/api/chat" });

<CrayonChat chat={chat} agentName="My Agent" />
```
