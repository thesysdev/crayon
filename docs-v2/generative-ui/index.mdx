# Generative UI

Generative UI is the core of Crayon. Instead of the LLM returning markdown that you render as text, it returns structured output that Crayon renders as real React components.

## How it works

1. **You register components** — define your React components and describe their props with zod schemas in a component registry.
2. **Crayon generates a schema** — the registry produces a format that tells the LLM what components are available and what props they accept.
3. **The LLM responds in GenUI Lang** — a subset of JavaScript designed for LLM output. The LLM composes your registered components.
4. **The renderer displays it** — Crayon's renderer parses the LLM response (including partial/streaming output) and renders the matching components.
