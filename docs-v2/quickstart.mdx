---
title: Quick Start
description: Get a working Crayon app in 30 seconds.
---

```bash
npx create-crayon-app my-app
cd my-app && pnpm dev
```

This scaffolds a full-stack app with:
- A pre-built chat UI powered by `CrayonChat`
- Generative UI rendering via component registry
- A backend with LLM integration

Open your browser and start chatting.

## What's in the scaffold

```
my-app/
├── app/
│   ├── page.tsx              # CrayonChat + useChat setup
│   └── api/chat/route.ts     # LLM endpoint with streaming
├── components/               # Registered GenUI components
├── registry.ts               # Component registry with zod schemas
└── package.json
```
